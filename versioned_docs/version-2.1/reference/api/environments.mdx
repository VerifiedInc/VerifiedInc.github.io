### Mock Data

In Sandbox, we return mock user data regardless of what phone (or email) [identifier](/reference/data/outputs#identifiers) you use. 

:::tip Use Your Own Phone Number
**We recommend that you use own phone number,** so you can succesfully receive SMS messages during testing. We return mock data for any phone number in Sandbox (other than those for specific [test users](/1-click-signup/test-users) designed to not return data).
:::

:::tip Use Our Test Users
**We recommend that you use our [test users](/1-click-signup/test-users),** which cover a wide variety of different test cases.
:::

If you'd like us to return custom mock data for your phone number(s), please email us at Support@Verified.inc â€” we're happy to enable that for you.
