If you don't include `credentialRequests` in the request body, the [default credential requests](/guides/setup#default-credential-requests) setting for your brand will apply. For reference, the standard credential requests (see [here](/guides/setup#default-credential-requests) in the Setup guide) include the [core KYC data points](/data#kyc-know-your-customer), all set to optional:
