Append the `/v2` path to the Sandbox base URL for your Verified API calls, and replace your old Sandbox API key with the v2 Sandbox API key for your brand. _You can copy this API key from the top of the Brand Details page in the Dashboard._

In v1, you had to include your API key as a bearer token in the authorization header. In v2, you should include your API key directly, not as a bearer token: just remove `Bearer` from the authorization header.

    <tr>
        <th>Version</th>
        <th>Base URL</th>

    </tr>
    <tr>
        <th>v1</th>
        <td><code><span>https://</span>core-api.sandbox-verifiedinc<b>.com</b></code></td>

    <tr>
        <th>v2</th>
        <td><code><span>https://</span>core-api.sandbox-verifiedinc<b>.com</b>/v2</code></td>
        <td><code>Authorization: `{API_KEY}`</code></td>
    </tr>
</table>
