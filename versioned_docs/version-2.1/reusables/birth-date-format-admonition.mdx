<Admonition type="info" title="Birth Date Format">
    You'll need to pass the birth date to our API (in the [request body of `POST /1-click`](/reference/api/endpoints#post-1-click-request)) in `YYYY-MM-DD` format, but you should collect it from the user in whatever form you think results in the best user experience for your use case.
</Admonition>
