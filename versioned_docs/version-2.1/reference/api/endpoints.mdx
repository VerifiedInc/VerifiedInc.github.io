### `POST /1-click/verifications`

> Begin a verification flow

<table>
  <tr>
    <th>Method</th>
    <td>`POST`</td>
  </tr>
  <tr>
    <th>Path</th>
    <td>`/1-click/verifications`</td>
  </tr>
</table>


#### Request {#post-1-click-verifications-request}

```typescript title="POST /1-click/verifications Request Body"
{
    channel: "autofill" | "sms" | "email",
    deviceIp?: string, // only relevant if using autofill channel
    phone?: string // only relevant if using SMS channel or if wanting to match user input to autofill output
}
```

<table>
    <tr>
        <th>Property</th>
        <th>Required?</th>
        <th>Type</th>
        <th>Format</th>
        <th>Default</th>
        <th>Description</th>
        <th>Example</th>
    </tr>
    <tr>
        <td>`channel`</td>
        <td><Chip label="Required" variant="outlined" size="small" color="error"/></td>
        <td>
            <details>
                <summary>`enum`</summary>

                Possible values:
                    - `"autofill"`
                    - `"sms"`
                    - `"email"` <Chip label="Coming Soon" variant="outlined" size="small" color="primary"/>
            </details>
        </td>
        <td>`snake_case`</td>
        <td>-</td>
        <td>Channel used for verification</td>
        <td>`"autofill"`</td>
    </tr>
    <tr>
        <td>`deviceIp`</td>
        <td><Chip label="Optional" variant="outlined" size="small" color="primary"/> (but required if `channel` is `"autofill"`)</td>
        <td>`string`</td>
        <td>[IPv6](https://en.wikipedia.org/wiki/IPv6_address)</td>
        <td>-</td>
        <td>IP address of user's device</td>
        <td>`"2001:0db8:85a3:0000:0000:8a2e:0370:7334"`</td>
    </tr>
    <tr>
        <td>`phone`</td>
        <td><Chip label="Optional" variant="outlined" size="small" color="primary"/> (but required if `channel` is `"sms"`)</td>
        <td>`string`</td>
        <td>[E.164](https://www.twilio.com/docs/glossary/what-e164)</td>
        <td>-</td>
        <td>User's phone number (in [E.164](https://www.twilio.com/docs/glossary/what-e164) format)</td>
        <td>`"+12125550010"`</td>
    </tr>
</table>


#### Response {#post-1-click-verifications-response}

```typescript title="POST /1-click/verifications Response Body"

    ...1ClickVerificationEntity // includes phone and verified status if autofill succeeds (and matches phone input if provided)
}
```

See [`1ClickVerificationEntity`](/reference/api/types#1clickverificationentity).

#### Request {#get-1-click-verifications-request}

