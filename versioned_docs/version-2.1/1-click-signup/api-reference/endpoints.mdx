    ## `GET /1-click`
</EndpointHeaderWithoutSwagger>

> Get data for a user who has completed a 1-Click Signup flow

<table>
  <tr>
    <th>Method</th>
    <td>`GET`</td>
  </tr>
  <tr>
    <th>Path</th>
    <td>`/1-click/{identityUuid}`</td>
  </tr>
</table>

<IntegrationTypeEndpointAdmonition option="SDK"/>


### Request {#get-1-click-request}

Call:
```
GET /1-click/{identityUuid}
```
- For the `{identityUuid}` path parameter, use the value of the `identityUuid` returned by the SDK or included as a URL parameter on the redirect URL.


### Response {#get-1-click-response}

```typescript title="GET /1-click Response Body"
{
    ...1ClickEntity
}
```

See [`1ClickEntity`](../api-reference/types#1clickentity).
