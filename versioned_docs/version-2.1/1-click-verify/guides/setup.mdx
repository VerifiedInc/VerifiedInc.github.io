<Admonition type="warning" title="The Second Option Requires Approval">
    If you choose the second option, we must approve you for [Production](/reference/api/environments#production) access by evaluating that you're properly verifying users' phone numbers.
</Admonition>

If you choose the first option, you're done! There's no need to do any integration work, because the SDK handles everything for you.

If you choose the second option, follow the 1-Click Verify [API Integration](/1-click-verify/guides/api-integration) guide.
