You can use any of the [1-Click Signup test users](/1-click-signup/test-users) with 1-Click Verify. 

No SMS will be sent to the test user phone numbers, since they aren't real phone numbers. Unless otherwise noted, you can verify for any test user using the static verification code **111111**.
