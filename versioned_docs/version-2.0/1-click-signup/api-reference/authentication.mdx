All requests to the Verified API require an API key as an `Authorization` header value. 

You can access and manage the API keys for your brands in the Verified [Dashboard](https://dashboard.verified.inc).

<ApiKeysAdmonition/>

Your company account can have one or more brands, and each brand can have one or more API keys. Each API key is specific to a brand and an [environment](./environments).
