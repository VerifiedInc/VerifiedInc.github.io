<Admonition type="info" title="Supported Phone Numbers">
    **Currently, we only support US mobile phone numbers.** You'll need to pass these to our API in [E.164 format](https://www.twilio.com/docs/glossary/what-e164). This is the `+1` country code with a 10-digit subscriber code appended to it, for example `+12125550010`.
</Admonition>
