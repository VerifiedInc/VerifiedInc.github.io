:::tip Risk Signals Enable Multiple Addresses
**We can return up to 3 most recent [addresses](/reference/data/outputs#core-data) for a user,** if you have risk signals turned on (set to **Basic** or **Advanced**). We order these addresses from most to least recent, as best as we're able to determine recency. 

You can show the user a dropdown to select one of the addresses on the info confirmation step (see [here](/1-click-signup/user-experience#info) on the User Experience page). We highly recommend this because it helps reduce the chance the user will need to manually enter an address, reducing friction and boosting conversion.
:::
