## Install with NPM

To install the SDK with NPM, run:

```
npm i @verifiedinc-public/client-sdk
```


## Install with Script

```html
<script src="https://cdn.jsdelivr.net/npm/@verifiedinc-public/client-sdk@1.3.1/dist/index.umd.min.js" type="module"></script>
```
