[Phone identifiers](/data#identifiers) (which are passed as inputs to the Verified API) must be in [E.164](https://www.twilio.com/docs/glossary/what-e164) format: for example, `"+12125550010"`. 1-Click Signup currently only supports US phone numbers (ones that start with country code `+1`).
