## 2. Configure brand settings.

**On the Brands page, click the Create button to create a new brand, or click Brand Details to access brand settings for an existing brand.** 

:::info
If you create a new brand, you can configure the [integration type](#integration-type), [phone verification](#phone-verification), and [risk signals](#risk-signals) settings in the brand creation dialog. **These settings are only relevant for [1-Click Signup](/1-click-signup), and any selections will work for [Text to Signup](/text-to-signup).** When you click the Create button to create the brand, you'll be taken to the its Brand Details page, where you can configure the remaining settings.
:::
